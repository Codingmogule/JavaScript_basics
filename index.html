Why use JAVASCRIPT

.It enhances user experience by enabling developers to create interactive and dynamic webpages.
.It is ideal for client side interactivity since it runs on web browsers.
-It also enables developers to modify the content of the webpage easily without realoading the whole page.
-This makes interactivity faster and better.
JavaScript is compatible with major web browsers e.g Firefox,Google,Microsoft edge....making it more efficient.

Variables and data types..

A variable is basically a container that can be used to store and manipulate data during the execution of a program.

There are different types of variables which are...

  • String - This is basically a collection of characters.
  • Numbers - This represents the numerical values(integers,float numbers)
  • Boolean - This represents the true or false values.
  • Null - This represents the absence of a value.
  • Undefined - This represents the absence of a value.
  • Array - This is basically a connection of values represented by square brackets[]
  • Object - A variable that has key value attributes.

Naming of variables..

You can either use..
.Camel case
.Underscore
.Don't use spaces
.Don't start with a number.
.Don't use javascript keywords.
